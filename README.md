# 🎬 Netflix Data Analysis

> An Exploratory Data Analysis (EDA) project on the Netflix Movies & TV Shows dataset using Python.

## 📌 Project Overview

This project analyzes the Netflix dataset to discover trends in movies and TV shows. It explores content distribution, ratings, release years, genres, countries, and other insights using data visualization techniques.

## 📊 Objectives

- Clean and preprocess the dataset
- Analyze Movies vs TV Shows
- Explore content ratings
- Find top countries producing Netflix content
- Analyze release year trends
- Identify popular genres
- Visualize insights using charts

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
